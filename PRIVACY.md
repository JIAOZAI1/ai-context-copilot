# Privacy Policy

**Effective Date: May 26, 2026**

Your privacy is extremely important to us. This Privacy Policy describes how our Chrome Extension (the "Extension") handles user data.

## 1. Data Collection and Usage
- **No Personal Data Collection:** The Extension does not collect, store, or transmit any of your personal identification data, browsing history, or personal information.
- **Local Storage Only:** All configuration settings, such as your self-provided API Keys, custom Base URLs, and model preferences, are stored strictly on your local device using Chrome's secure `chrome.storage.local` API. We have no access to this data.
- **In-Context Text Processing:** The Extension only reads the webpage text that you actively select with your mouse. This text is strictly used as context to send directly to the official LLM API endpoints (e.g., OpenAI, DeepSeek) that you configure. 

## 2. Third-Party Services
The Extension communicates directly from your browser to the LLM API providers you choose. Please review the privacy policies of your respective API providers (such as OpenAI or DeepSeek) regarding how they handle API request data. No data is routed through any intermediate or third-party servers controlled by us.

## 3. Changes to This Policy
We may update our Privacy Policy from time to time. Any changes will be posted on this page.

## 4. Contact Us
If you have any questions about this Privacy Policy, please contact us via our GitHub Repository Issues page.

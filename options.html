100% 隐私保护承诺

本插件 不依赖任何自建后端服务器，所有代码均为纯前端驱动。

  • 你的 API Key 仅存储在浏览器本地 chrome.storage.local 中
  • AI 请求直接在你的浏览器与 API 服务商之间点对点通信
  • 不经过任何中间代理或第三方服务器
  • 不会收集、上传或分享你的浏览数据、选中文本或对话内容
  • 所有源代码均可审查，无混淆、无远程代码注入
你的数据，永远属于你。
